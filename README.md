# Módulo - Webpack

## Instalação

1. Iniciar o projeto no diretório
```
npm init -y
```

2. Instalar o Webpack como dependência de desenvolvimento
```
npm install --save-dev webpack webpack-cli
```

## Trabalhando com HTML
```
npm install --savedev html-webpack-plugin
```
